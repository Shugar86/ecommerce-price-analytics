### ✅ ШАГ 3: Распределенный сборщик данных
**Статус:** Завершен на 100%

**Созданный файл:**
- ✅ `app/collector.py` (362 строки)

**Реализованные функции ETL:**

1. **`fetch_currency(session)`** - Сбор курсов валют
   - Источник: ЦБ РФ XML API
   - Кодировка: windows-1251
   - Парсинг: lxml.etree
   - Операция: UPSERT курса USD

2. **`fetch_foreign_goods(session)`** - Зарубежные товары
   - Источник: FakeStore API (JSON)
   - Получение курса USD из БД
   - Конвертация цен: price_usd * usd_rate
   - Операция: UPSERT товаров

3. **`fetch_russian_goods(session)`** - Российские товары
   - Источник: TBM Market YML
   - **Потоковый парсинг:** iterparse() + очистка памяти
   - Обработка: первые 20 товаров (demo Big Data)
   - Операция: UPSERT товаров

4. **`collect_all_data(session)`** - Полный цикл сбора
5. **`main()`** - Бесконечный цикл с обновлением каждый час

**Особенности реализации:**
- ✅ Streaming XML parsing (memory-efficient)
- ✅ PostgreSQL UPSERT через ON CONFLICT DO UPDATE
- ✅ Try/except блоки с конкретными исключениями
- ✅ Структурированное логирование
- ✅ Автоматический retry при ошибках БД

**Диаграмма:**
- ✅ DFD в Gane-Sarson нотации (README_REPORT.md, раздел 3.3)
